HTML tags and their description

Here are 5 HTML tags that I learned.
body tag
the main body of our document, all the content of the web page must be added inside this tag.
main tag
primary content of the web page.
img tag
this tag is used for displaying images on our web pages.
a tag
for creating a link on our web page, we use the anchor tag
form tag
we use this tag to communicate and interact with our client and collect various types of data.
